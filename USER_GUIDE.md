# Guide Utilisateur — Routine

## Premiers pas

Au lancement, l'écran d'accueil propose deux modes :

- **Enfant** — accès direct aux routines
- **Parent** — espace de gestion protégé par PIN

---

## Espace Parent

### Créer un PIN

Au premier accès parent, vous définissez un code PIN à 4 chiffres. Ce code protège l'espace de configuration pour que les enfants ne modifient pas les routines.

### Ajouter un enfant

1. Aller dans **Espace Parent**
2. Appuyer sur **+ Ajouter** dans la section Enfants
3. Remplir : nom, âge, avatar (emoji au choix), couleur
4. Sauvegarder

Vous pouvez modifier un profil en appuyant dessus depuis le dashboard.

### Créer une routine

1. Appuyer sur **+ Créer** dans la section Routines
2. Choisir l'enfant concerné
3. Donner un nom, une icône, une couleur
4. Choisir une catégorie : Matin, Soir, École, Maison, Week-end ou Personnalisé
5. Ajouter des étapes :
   - **Titre** : ce que l'enfant doit faire (ex: "Brosser les dents")
   - **Icône** : emoji représentatif
   - **Durée** : temps estimé en minutes (0 = pas de timer)
   - **Consigne** : instruction facultative
   - **Obligatoire/Facultatif** : les étapes facultatives sont ignorées si l'humeur est négative
6. Réorganiser les étapes si besoin
7. Sauvegarder

### Utiliser le catalogue

Appuyer sur **Catalogue** pour accéder à 20 modèles de routines prêts à l'emploi (matin, soir, école, hygiène, week-end...). Sélectionnez un modèle, choisissez l'enfant, et la routine est créée automatiquement. Vous pouvez la modifier ensuite.

### Activer / Désactiver une routine

Depuis le dashboard, chaque routine a un interrupteur. Une routine désactivée n'apparaît plus côté enfant.

### Partager une routine

1. Aller sur une routine → **Partager**
2. Un lien encodé est généré
3. Copier et envoyer par messagerie
4. Le destinataire ouvre le lien → import automatique

### Importer une routine

1. Appuyer sur la tuile **Importer**
2. Coller le lien reçu
3. Choisir l'enfant cible
4. La routine est dupliquée et modifiable

### Récompenses réelles

1. Appuyer sur la tuile **Récompenses**
2. Créer une récompense : description + nombre d'étoiles requis
3. Quand l'enfant atteint le nombre d'étoiles, la récompense est débloquée
4. Exemples : "Sortie au parc", "Choisir le dessert", "30 min de jeu vidéo"

### Statistiques

La tuile **Statistiques** montre par enfant :
- Nombre de routines complétées
- Série en cours et record
- Historique jour par jour

### Réglages météo

Dans la section **Météo** du dashboard parent :
- **Géolocalisation automatique** : active le GPS pour détecter la ville (l'appareil demande la permission)
- **Ville manuelle** : tapez le nom de votre ville (ex: Rennes, Lyon, Bourges) et validez

La météo s'affiche ensuite sur l'écran de l'enfant avec un fond adapté, une tenue conseillée et des messages ludiques.

## Espace Enfant

### Choisir son profil

Si plusieurs enfants sont configurés, l'enfant appuie sur son avatar pour accéder à ses routines. S'il n'y a qu'un seul profil, la sélection est automatique.

### Écran d'accueil

L'écran principal affiche :
- **La météo** avec un dégradé de fond adapté (jour/nuit, pluie, soleil...)
- **Un message d'accueil** selon le moment de la journée
- **La tenue conseillée** (icônes : chaussettes, veste, lunettes...)
- **Les routines actives** sous forme de grosses cartes colorées
- **La récompense en cours** avec une barre de progression

### Lancer une routine

1. Appuyer sur une carte routine → **"Go !"**
2. Choisir son humeur (content, motivé, fatigué, en colère...)
3. L'exécution commence étape par étape

### Exécuter les étapes

Pour chaque étape :
- L'icône et le titre s'affichent en grand
- Un **timer circulaire** décompte la durée (si une durée est définie)
- Un message d'encouragement s'affiche
- Appuyer sur **"C'est fait !"** pour passer à l'étape suivante
- Les étapes facultatives peuvent être passées avec **"Passer cette étape"**

La barre de progression en haut montre l'avancement global.

### Célébration

À la fin de la routine :
- Animation de confettis et particules colorées
- Étoiles gagnées affichées en grand
- Badges débloqués annoncés
- Bouton pour revenir à l'accueil

### Voir ses récompenses

Appuyer sur le compteur d'étoiles pour voir :
- **Total d'étoiles** accumulées
- **Série en cours** (jours consécutifs)
- **Record de série**
- **Badges débloqués** et verrouillés
- **Récompenses réelles** et progression

### Routine calme / Bien-être

La bannière "Routine calme" donne accès à des exercices de respiration et d'étirement, idéaux avant le coucher.

## Météo adaptative

L'écran enfant s'adapte automatiquement à la météo :

- **Fond dégradé** qui change selon la condition (soleil, pluie, neige, orage, nuit...)
- **Particules flottantes** (flocons, gouttes, étoiles...)
- **Message ludique** adapté aux enfants ("Les nuages jouent à cache-cache !")
- **Tenue conseillée** :
  - **Jour** : vêtements adaptés à la température + météo (parapluie quand il pleut, lunettes au soleil...)
  - **Nuit** : kit dodo (pyjama, doudou, livre, ours en peluche) + ajustements selon la température
- **Conseil pratique** ("N'oublie pas ton parapluie ☔")

---

## Astuces

- **Commencez simple** : 3-4 étapes par routine suffisent pour un jeune enfant
- **Utilisez le catalogue** : les modèles prêts à l'emploi sont un bon point de départ
- **Encouragez les séries** : montrez à l'enfant son compteur de flamme
- **Récompenses réelles** : associez des étoiles à de petites récompenses pour motiver
- **Humeur négative** : quand l'enfant est fatigué ou grognon, les étapes facultatives sont automatiquement retirées pour simplifier la routine
